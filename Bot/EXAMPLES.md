## Tips for Great Stories

### Do's ‚úÖ
- **Build on previous contributions**: Read what others wrote
- **Stay consistent**: Keep tone and style similar
- **Add detail**: Include sensory details and emotions
- **Move forward**: Progress the plot with each addition
- **Use dialogue**: Characters speaking adds life

### Don'ts ‚ùå
- **Don't contradict**: Respect what's already established
- **Don't end abruptly**: Let others continue (unless using !story end)
- **Don't write novels**: Keep it to 1-2 sentences per turn
- **Don't ignore context**: Make sure your addition makes sense

## Command Cheat Sheet

| Command | When to Use |
|---------|-------------|
| `!story start` | Begin a new story |
| `!add <sentence>` | Continue the story |
| `!story show` | Review what's been written |
| `!story summary` | Get caught up on a long story |
| `!story image` | Visualize the current scene |
| `!story end` | Conclude with an AI ending |

## Collaborative Tips

### For 2-3 Players
- Take turns naturally
- Build suspense together
- Plan climactic moments

### For Groups (4+)
- Use `!story summary` every 5-7 contributions
- Designate someone to guide the plot
- Consider themed story nights

### Solo Practice
- Write multiple contributions yourself
- Experiment with different genres
- Test the AI's summary capabilities
